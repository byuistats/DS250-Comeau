Pull and Merge Forks on GitHub

Create Pull Request

  1. Go the the forked repository in byuids-resumes and click Pull request.

  1. This will bring you to the the following page where you need to click switching the base.

  1. Now you can Create pull request.

  1. Here you can type a note and then actually Create pull request.

  1. Now you need to View pull request.

Merge Request

If you have admin access of the forked repository where you are doing the pull request, you can finish the next two steps.

  1. Click the Merge pull request button.

  1. Now confirm the merge.
